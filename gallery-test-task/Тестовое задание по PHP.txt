Здравствуйте. Высылаю Вам тестовое задание по PHP.

Создать галерею для просмотра картинок.
Галерея должна поддерживать: 
  
1. Вывод всех картинок, желательно уменьшенных. Под картинкой должно быть написано имя файла. При клике на картинку она должна открываться в полном размере, можно в отдельном окне. 
  
2. Загрузку картинок пользователем. При загрузке должна быть возможность ввести теги. Не-картинки не должны попадать в галерею. Можно вводить несколько тегов.
  
3. Вывод картинок по заданному тегу.
  
Желательно: 
  
4. Реализовать API для получения информации о картинке по её ID в формате JSON.  
Информация должна содержать имя файла, его размер в байтах, размеры картинки в пикселях и ссылку на картинку. Форма запроса произвольная. 
Пример 
запрос: http://gallery.local?picid=13 
ответ: {"id":"13","filename":"serenity.jpg","size":"21315","width":"1024","height":"768","tags":["природа", "облака"]} 
  
5. Реализовать API для получения информации по существующим тегам и количеству картинок в них, в JSON. 
Пример 
запрос: http://gallery.local?tags 
ответ: {"облака":"12","лошади":"3","коты":"2423794353"} 
  
Никаких "красивостей" можно не делать, нас интересует только PHP-код. 
Должна быть обеспечена масштабируемость - т.е. должна быть возможность относительно быстро добавить поддержку категорий, изменить способ хранения файлов и т.п.
